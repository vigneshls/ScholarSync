# 🎓 ScholarSync

> **Orchestrating Academic Excellence with Heroic Precision.**

ScholarSync is a high-performance, desktop-native academic management hub designed for the modern educational administrator. Built with a premium **Glassmorphism UI**, it transforms institutional data into a live, actionable command center.

---

## 🌟 Key Features

### 🏛️ Live Command Center
- **Dynamic Task Distribution**: Real-time tracking of CSE, AI & DS, and IT department workloads.
- **Unit Efficiency Analytics**: Immediate visibility into planned, in-progress, and completed academic tasks.
- **Urgent Priorities**: A specialized "Critical Task" sentinel that ensures deadlines are never missed.

### 🎨 Heroic Aesthetics & Themes
ScholarSync features a suite of high-fidelity themes to match your work style:
- **Scholarly Dark & Academic Light**: Professional standard modes.
- **Night Knight (Batman)**: A high-contrast vigilante aesthetic.
- **Man of Steel (Superman)**: Heroic blue and red.
- **Wall Crawler (Spider-Man)**: Dynamic web-styled UI.
- **Gamma Rage (Hulk)** & **Silent Assassin (Hitman)**: Specialized palettes for deep focus.

### 📊 Institutional Intelligence
- **Demographic Breakdown**: Visualized student strength (Male/Female) and year-wise body counts.
- **Faculty Performance Tracking**: Comprehensive monitoring of faculty workload and task history.
- **Academic Standardized**: Built-in compatibility for academic terminology (Faculty, Tasks, IT/CSE labels).

## 🚀 Getting Started

### Prerequisites
- Node.js (Latest stable version)
- npm or yarn

### Installation (Development)
1. Clone the repository:
   ```bash
   git clone https://github.com/vigneshls/ScholarSync.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the application:
   ```bash
   npm run start
   ```

### 📦 Build & Installer
The standalone production installer for ScholarSync is protected. To obtain the official `.exe` for your institution, please contact the repository owner directly.

## 🛠️ Technology Stack
- **Frontend**: React.js with Vite
- **Shell**: Electron.js
- **Styling**: Vanilla CSS (Custom Glassmorphism framework)
- **Charts**: Recharts
- **Icons**: Lucide React
